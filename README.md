# Clinic CRM

これはクリニック向けの顧客関係管理（CRM）アプリケーションです。患者とスタッフの両方に機能を提供します。

## 主な機能

### 患者向け機能
- **認証**: 患者はログインして自分のアカウントにアクセスできます。
- **予約管理**:
    - 空き状況を確認して、施術やサービスの予約ができます。

### スタッフ向け機能
- **専用ダ-ッシュボード**: スタッフ専用の管理画面。
- **患者管理**: 患者情報の登録、表示、更新、削除（CRUD）ができます。
- **契約管理**:
    - 患者との契約を作成・管理できます。
    - 契約の使用状況を追跡できます。
- **スタッフ管理**: スタッフメンバーのアカウントを管理できます。
- **リソース管理**:
    - **メニュー管理**: 施術メニューやサービスのCRUD。
    - **商品管理**: 販売する商品のCRUD。
    - **部屋管理**: クリニック内の部屋のCRUD。
    - **機器管理**: 医療機器や設備のCRUD。
- **スケジュール管理**:
    - **シフト管理**: スタッフのシフトを管理できます。
    - **予約管理**: 患者の予約を一覧で確認・管理できます。
- **ドキュメント管理**:
    - 同意書などのドキュメントテンプレートを管理できます。
    - 患者からデジタル署名を取得できます。
- **監査ログ**: スタッフが行った操作のログを確認できます。
